# rshell
## Overview
This is *rshell*. *rshell* is a pseudo terminal that runs bash commands and scrips.

## How to Build
```bash
$ git clone https://github.com/KojinKuro/rshell.git
$ cd rshell
$ git checkout exec
$ make
$ ./bin/rshell
```
